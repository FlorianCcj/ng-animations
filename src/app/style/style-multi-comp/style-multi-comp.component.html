Un style générale peut s inclure de 2 manieres
  • soit dans l angular-cli.json dans l'attribut style ;
  • soit dans l'index.html comme dans un html classique, avec une balise link
View Encapsulation (attribute of @Component)
  • encapsulation: ViewEncapsulation.Emulated: (default) just for the component
  • encapsulation: ViewEncapsulation.Native: Uses natuve Shadow DOM
  • encapsulation: ViewEncapsulation.None: Styles are shared accross Component
